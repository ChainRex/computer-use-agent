## 🚀 部署说明

### Linux服务端
```bash
python server/main.py
# 服务端地址: ws://localhost:8000/ws
```

### Mac客户端
1. 下载 `computer_use_agent_client.zip`
2. 解压并安装依赖: `pip install -r requirements.txt`
3. 启动客户端: `python start_client.py`
4. 连接服务端（修改IP地址为Linux服务端地址）
